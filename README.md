# Skill Flow - Final Year Project (FYP)

## Introduction
Skill Flow is a professional freelance marketplace and skill-sharing platform designed to bridge the gap between talented freelancers and clients looking for high-quality services. This project was developed as a Final Year Project (FYP) for PMAS-AAUR in 2025.

## Goal
The primary objective of Skill Flow is to provide a seamless, secure, and efficient ecosystem where users can:
- Showcase their professional skills.
- Connect with potential clients.
- Collaborate on projects using integrated communication tools.
- Handle payments securely.
     
- ## Tech Stack
Skill Flow is built using the **MERN** stack along with several modern integrations:
    
- ### Frontend
- **React (Vite)**: For a fast and responsive user interface.
- **Tailwind CSS**: For modern and scalable styling.
- **CometChat & Stream Chat**: Integrated for real-time messaging and collaboration.
- **Stripe**: For secure payment processing.
- **Google Generative AI**: Leveraging AI for enhanced platform features.
- **Lucide React**: For beautiful iconography.
                 
- ### Backend
- **Node.js & Express**: A robust server-side architecture.
- **MongoDB & Mongoose**: A flexible document-based database.
- **JWT & Bcrypt**: For secure authentication and data protection.
- **Nodemailer**: For automated email communications.
- **Multer**: For handling file uploads.
                           
- ## Future Scope
- **AI-Driven Job Matching**: Implementing smarter algorithms to recommend the best freelancers for specific tasks.
- **Advanced Analytics**: Providing users with detailed insights into their performance and earnings.
- **Mobile Application**: Developing dedicated iOS and Android apps for on-the-go access.
- **Multi-Currency Support**: Expanding payment options to support a global user base.
